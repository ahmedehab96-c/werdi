# Werdi Quran App

## About

Werdi (وردي) is an Arabic-first Quran memorization and revision app with full English support. It helps users memorize ayah by ayah, review saved verses, browse the Mushaf with tafsir and bookmarks, and practice voice recitation (tasmee3) with automatic feedback.

The app features dark/light/system themes, multiple reciters, offline storage with Drift, optional Supabase sync, and a responsive layout for phones and tablets. Built with Clean Architecture, BLoC, and go_router.

وردي (Werdi) هو تطبيق عربي أولاً لحفظ ومراجعة القرآن الكريم مع دعم إنجليزي كامل. يساعد المستخدم على الحفظ آية بآية ومراجعة المحفوظ وتصفح المصحف مع التفسير والإشارات المرجعية والتسميع الصوتي مع تصحيح تلقائي.

يتضمن التطبيق ثيمات نهاري/ليلي/تلقائي وعدة قراء وتخزين محلي عبر Drift ومزامنة اختيارية مع Supabase وتخطيط متجاوب للهواتف والأجهزة اللوحية. مبني بـ Clean Architecture وBLoC وgo_router.

## Features

- Ayah-by-ayah memorization sessions with repetition controls
- Voice recitation (tasmee3) flow with automatic correction
- Smart revision list and progress tracking
- Quran browsing with search, tafsir, and bookmarks
- Dark, light, and system theme modes
- Responsive UI for small phones and tablets
